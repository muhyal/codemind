# codemind

A macOS application built with SwiftUI.

## Features

*   Global shortcut (Cmd+Shift+0) to toggle the main window.

## Getting Started

1.  Clone the repository.
2.  Open `codemind.xcodeproj` in Xcode.
3.  Build and run the project.

## Technology Stack

*   Swift
*   SwiftUI
*   SwiftData
*   AppKit
